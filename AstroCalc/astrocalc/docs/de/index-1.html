1. Einführung

AstroCalc ermöglicht die Berechnung diverser Teleskop-Spezifikationen, wie z.B. Auflösungsvermögen, Grenzgröße etc. Desweiteren stehen einige Tools (2 in der Version 0.3) für die Ermittlung des Julianischen Datums und zur Berechnung des Helligkeitsunterschiedes zweier Objekte (Sterne, Nebel etc.).

1.1 Änderungen

  • + 2 Tools (Juliansches Datum, Mag-Differenz)
  • + Änderung der GUI
