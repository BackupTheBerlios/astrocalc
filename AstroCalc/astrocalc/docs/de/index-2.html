2. Installation

2.1 Wie erhält man AstroCalc

Die aktuellste Version kann immer bei SourceForge.net heruntergeladen werden: http://www.sourceforge.net/projects/sternensaal/.

2.2 Anforderungen

  • automake, autoconf etc.
  • X11
  • QT2 (incl. der Header-Dateien)

2.3 Kompilierung und Installation

Um AstroCalc zu kompilieren und zu installieren sind folgende Schritte im Hauptverzeichnis der AstroCalc Distribution notwendig: 

% ./configure
% make
% make install

Da AstroCalc 

autoconf
nutzt, sollten während des Kompilierens keine Schwierigkeiten auftreten.

Bei Problemen bitte eine möglichst genaue Fehlerbeschreibung an den Autor Michael Cebak schicken.
